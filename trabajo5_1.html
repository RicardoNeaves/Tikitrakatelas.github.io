Etiquetas de HTML5

A continuacion se inidicaran las etiquetas que utilizamos para cada seccion de nuestra pagina web acerca de las carreras de FIME.


Etiquetas iniciales

Estas son las etiquetas que van al inicio de nuestro codigo.

  • !DOCTYPE html Indica al navegador que el documento está basado en el estándar HTML5
  • html /html Representa la raíz de un documento HTML. Todos los demás elementos de la estructura HTML deben ser recogidos dentro de estas etiquetas.

    • Etiquetas de documento

  • head /head Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
  • title /title Esta etiqueta la utilizamos para definir el título de la página web: "Trabajo 5".

    • Etiquetas de secciones

  • body /body Adentro de esta etiqueta, recogemos todo lo que queremos mostrar en nuestra pagina web.
  • h1, h2, h3. h4. h5. h6 Estas etiquetas las utilizamos para darle diferente tamaño a los titulos en la pagina web, asi podemos jerarquizar la informacion de una manera mas facil.

    • Etiquetas para agrupacion de contenido

  • p /p Esta etiqueta la utilizamos para escribir y separar entre parrafos de texto.
  • hr La utilizamos como separador, para romper entre secciones de nuestra pagina y asi tener espacios.
  • ol /ol Etiqueta para crear una lista ordenada
  • li /li Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.

    • Etiquetas para insertar multimedia

  • img /img Etiqueta para insertar una imagen en la página web.
  • video /video Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.
  • audio /audio Usada para cargar en una web un archivo de audio o stream de audio.
